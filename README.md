## Objective
To develop an interactive Excel dashboard to analyze and monitor sales performance across cities, customer types, and product categories, helping management make data-driven decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Slicers
- Power Query (Data Cleaning)
- Excel Formulas (SUM, AVERAGE, IF)
- Conditional Formatting

## Key Performance Indicators (KPIs)
- Total Sales Revenue
- City-wise Sales (Chennai, Coimbatore, Madurai)
- Category-wise Sales (Chicken, Fish, Meat, Veg)
- Wholesale vs Retail Sales Contribution
- Average Product Price

## Dashboard Features
- KPI summary cards for quick performance overview
- Bar chart for city-wise sales comparison
- Pie/Doughnut chart for wholesale vs retail analysis
- Category-wise sales visualization
- Interactive slicers for City, Customer Type, and Category
- Dynamic updates based on user selection

## Configuration Details
- Cleaned and structured raw sales data using Power Query
- Created calculated column: `Total Price = Quantity Ã— Unit Price`
- Built Pivot Tables for aggregations
- Connected slicers to all charts using Report Connections
- Designed professional layout with consistent formatting
- Protected dashboard sheet for controlled access

## Conclusion
The AK Wholesale Market Sales Analysis Dashboard provides a clear and interactive view of business performance. It enables quick analysis of sales trends, customer segments, and city-wise revenue, supporting effective business decision-making.
