## Project Objective
To analyze wholesale market sales data and monitor performance trends using Excel.

## Tools Used
- Microsoft Excel

## Key Features
- Total Sales, Total Profit, Total Orders calculation
- Product Category Analysis
- Region-wise Performance
- Interactive charts and tables
- Conditional formatting for better visualization

## Data & Configuration Details
- Dataset organized in Excel sheets
- Pivot tables created for analysis
- Charts generated to visualize key metrics
- Calculations done using Excel formulas

## Conclusion
This Excel dashboard provides insights into wholesale market sales trends and supports informed business decisions.
